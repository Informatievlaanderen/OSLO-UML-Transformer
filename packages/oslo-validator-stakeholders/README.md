# OSLO Stakeholders Validator

> Validates stakeholder files for usage of OVO codes.

## Install

```bash
npm install @oslo-flanders/stakeholders-validator

```

## Global install

```bash
npm install -g @oslo-flanders/stakeholders-validator

```

## API

| Parameter     | Description                                                                             | Required           | Possible values  |
| ------------- | --------------------------------------------------------------------------------------- | ------------------ | ---------------- |
| `--input`     | The URL or local file path of an OSLO stakeholders JSON-LD file                          | :heavy_check_mark: |                  |

## Usage

```bash
oslo-jsonld-stakeholder --input stakeholders.jsonld
```
